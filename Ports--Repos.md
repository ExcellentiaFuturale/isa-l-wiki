## Ports-Repos

* FreeBSD [ports](http://www.freshports.org/devel/isa-l/)
    - To install the port: `cd /usr/ports/devel/isa-l/ && make install clean`
    - To add the package: `pkg install isa-l`
* [Clear Linux](https://clearlinux.org)
